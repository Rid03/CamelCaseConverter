# JSON Converter Java

## Описание
Программа `ConverterJava` предназначена для обработки JSON-файлов в указанной директории. Она автоматически проходит по всем файлам, конвертирует теги JSON в CamelCase и сохраняет изменения обратно в файлы.

## Функции
- Обход всех файлов в указанной директории.
- Обработка только файлов с расширением `.json`.
- Конвертация тегов JSON в CamelCase.
- Запись обработанных данных обратно в файлы.
- Вывод в консоль информации о успешно обработанных файлах.

## Требования
- Java 8 или выше.
- Библиотека Jackson для работы с JSON.

## Установка и запуск
1. Убедитесь, что у вас установлена подходящая версия Java.
2. Склонируйте репозиторий в желаемую директорию.
3. В файле `ConverterJava.java` измените путь к директории с вашими JSON-файлами.
4. Скомпилируйте и запустите программу.
